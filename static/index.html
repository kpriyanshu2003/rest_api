Home Page for https://www.api-uno.workers.dev

This is an api link which can do multiple functions. And they are described as follows :
GET Requests only

  1. / : Access home page
  2. /time : Get current time (Asia/Kolkata). Required parameter 'zone'
  3. /date : Get current date (Asia/Kolkata). Required parameter 'zone'
  4. /fact : Get a random fact
  5. /advice : Get a random advice
  6. /joke : Get a random joke (use bully if you want some fun)
  7. /quote : Get a random quote
  8. /bully : Get a random insult
  9. /greet : Greets the user. Optional parameter 'name'
  10. /convert : Convert binary, decimal, hexadecimal, and octal. Required parameter value, and type : 'bin', 'binary','dec','decimal','oct','octal','hex'or 'hexadecimal'.
