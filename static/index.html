Home Page for https://www.api-uno.workers.dev

This is an api link which can do multiple functions. And they are described as follows :
GET Requests only

  1. / : Access home page
  2. /time : Get current time (Asia/Kolkata). Accepts parameter 'zone'
  3. /date : Get current date (Asia/Kolkata). Accepts parameter 'zone'
  4. /fact : Get a random fact
  5. /advice : Get a random advice
  6. /joke : Get a random joke (not in service)
  7. /quote : Get a random quote
  8. /bully : Get a random insult
  9. /greet : Greets the user. Accepts parameter 'name'
  10. /convert : Convert binary, decimal, hexadecimal, and octal. Accepts parameter 'bin', 'dec', 'hex' and 'oct'. Single parameter
  11. /dictionary : Searches words in a dictionary. Requires parameter 'query'
